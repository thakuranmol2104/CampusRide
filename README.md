# 🚗 CampusRide

A carpooling app for college students to share rides and reduce commute costs.

## 🛠️ Tech Stack
- Android (Java)
- Firebase Auth
- Firebase Firestore (asia-south1)
- Firebase Cloud Messaging (FCM)

## 📁 Project Structure
```
app/
├── model/         # Data models (User.java, etc.)
├── helper/        # Firebase logic (FirebaseHelper.java)
├── ui/            # Activities & Fragments
└── res/           # Layouts & Resources
```

## 🚀 Getting Started
1. Clone the repo
2. Open in Android Studio
3. Add your `google-services.json` to the `app/` folder
4. Build & run

> ⚠️ Never commit `google-services.json` to the repo — it's in `.gitignore`
